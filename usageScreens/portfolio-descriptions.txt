# Lost & Found Management System - Portfolio Descriptions

## English Versions

### Option 1: The Short "Elevator Pitch" (Great for CVs or quick lists)
**Lost & Found Management System**
A comprehensive hybrid desktop and web application built to streamline the management of lost and found items. Powered by JavaFX with a flexible data layer designed for seamless switching between SQLite and PostgreSQL, the system features a secure desktop interface for administrators to manage records, handle image compression, and export data (PDF/CSV/Excel). ### Option 2: The Full Portfolio Case Study (Great for a dedicated project page)

#### **Project Title:** Lost & Found Management System

#### **Overview**
The Lost & Found Management System is a robust hybrid application designed to tackle the logistical chaos of managing lost items for organizations. It bridges the gap between secure, local desktop management and remote web accessibility. By combining a rich desktop client with an embedded web server, administrators can seamlessly track, manage, and return items from anywhere on the network, while users can access specific item statuses via QR codes. 

#### **Key Features**
*   **Dual-Interface Architecture:** Features a rich JavaFX desktop application for primary administration, alongside a mobile-friendly remote web dashboard.
*   **Database Agnostic Design:** Engineered with a flexible data layer that is fully compatible with both SQLite (for easy local setup) and PostgreSQL (for enterprise-grade scaling). 
*   **Smart Record Management:** End-to-end tracking of lost items, including status updates, return processing, and automated image compression for item photos.
*   **QR Code Integration:** Automatically generates unique QR codes for items, allowing secure public access to specific record statuses.
*   **Data Analytics & Export:** Includes a built-in statistics dashboard that tracks system usage. Administrators can export reports in multiple formats (Excel, CSV, PDF) for external audits.

#### **Technical Stack**
*   **Core:** Java 11+, Maven
*   **User Interface:** JavaFX (Desktop Client), HTML/CSS/JS (Web Dashboard)
*   **Database:** SQLite / PostgreSQL (Fully Compatible)
*   **Server:** Embedded Jetty Web Server (Port 8080)
*   **Key Integrations:** QR Code Generation, PDF/Excel Data Export, Base64 Image handling

#### **Technical Highlights & Challenges Solved**
*   **Scalable Data Architecture:** Built the application's database access layer to be entirely database-agnostic. While it runs on SQLite out-of-the-box for portability, the underlying structure allows for a frictionless migration to a PostgreSQL cluster for high-traffic deployments.
*   **Embedded Web Server Integration:** Successfully embedded a Jetty server within a standard Java desktop application, allowing the app to serve HTML traffic over the local network and internet (via port forwarding) without needing a separate backend service.

### Επιλογή 2: Αναλυτική Περιγραφή Project (Ιδανικό για τη σελίδα του Portfolio)

#### **Τίτλος Project:** Σύστημα Διαχείρισης Απολεσθέντων Αντικειμένων

#### **Σύνοψη**
Το Σύστημα Διαχείρισης Απολεσθέντων είναι μια στιβαρή υβριδική εφαρμογή σχεδιασμένη να επιλύει τις προκλήσεις διαχείρισης απολεσθέντων και ευρεθέντων αντικειμένων σε οργανισμούς. Συνδυάζοντας την ασφάλεια μιας εφαρμογής desktop με την ευελιξία της απομακρυσμένης πρόσβασης μέσω web, επιτρέπει στους διαχειριστές να παρακολουθούν, να καταχωρούν και να επιστρέφουν αντικείμενα από οποιοδήποτε σημείο του δικτύου, ενώ παράλληλα οι χρήστες μπορούν να ελέγχουν την κατάσταση των αντικειμένων τους μέσω κωδικών QR.

#### **Βασικά Χαρακτηριστικά**
*   **Υβριδική Αρχιτεκτονική:** Διαθέτει εφαρμογή JavaFX (desktop) για την κύρια διαχείριση και ένα mobile-friendly απομακρυσμένο web dashboard.
*   **Ευέλικτη Αρχιτεκτονική Δεδομένων:** Σχεδιασμένο με γνώμονα την επεκτασιμότητα, είναι πλήρως συμβατό με SQLite (για άμεση τοπική χρήση) και PostgreSQL (για απαιτήσεις εταιρικού επιπέδου).
*   **Έξυπνη Διαχείριση Αρχείων:** Ολοκληρωμένη παρακολούθηση αντικειμένων, ενημέρωση κατάστασης (status), διαδικασία επιστροφής και αυτόματη συμπίεση των φωτογραφιών τους.
*   **Ενσωμάτωση QR Codes:** Δεδομένης της ανάγκης για άμεση ενημέρωση, το σύστημα παράγει μοναδικά QR codes για κάθε αντικείμενο, επιτρέποντας την ασφαλή δημόσια πρόσβαση στην κατάσταση του εκάστοτε αρχείου.
*   **Αναλύσεις & Εξαγωγή Δεδομένων:** Περιλαμβάνει ενσωματωμένο dashboard στατιστικών που παρακολουθεί τη χρήση του συστήματος. Δυνατότητα εξαγωγής αναφορών σε πολλαπλά format (Excel, CSV, PDF) για εσωτερικούς ελέγχους.

#### **Τεχνολογίες Συστήματος (Tech Stack)**
*   **Core:** Java 11+, Maven
*   **User Interface:** JavaFX (Desktop Client), HTML/CSS/JS (Web Dashboard)
*   **Database:** SQLite / PostgreSQL (Πλήρης συμβατότητα)
*   **Server:** Ενσωματωμένος Jetty Web Server (Port 8080)
*   **Ενσωματώσεις:** Δημιουργία QR Code, Εξαγωγή PDF/Excel, Διαχείριση εικόνων (Base64/Compression)

#### **Τεχνικές Προκλήσεις που Επιλύθηκαν**
*   **Κλιμακούμενη Επίπεδο Δεδομένων (Data Layer):** Ανάπτυξη του επιπέδου πρόσβασης δεδομένων ώστε να είναι ανεξάρτητο από τη βάση (database-agnostic). Αν και λειτουργεί out-of-the-box με SQLite, η δομή του επιτρέπει την απρόσκοπτη μετάβαση σε PostgreSQL cluster για περιβάλλοντα με υψηλό φόρτο.
*   **Ενσωμάτωση Web Server:** Επιτυχής ενσωμάτωση ενός Jetty server εντός μιας τυπικής Java desktop εφαρμογής, επιτρέποντας στην εφαρμογή να εξυπηρετεί HTML traffic στο τοπικό δίκτυο (ή στο διαδίκτυο μέσω port forwarding) χωρίς την ανάγκη ανεξάρτητης backend υπηρεσίας.

Δημιούργησα μια υβριδική αρχιτεκτονική που εκτελεί μια JavaFX desktop εφαρμογή παράλληλα με έναν ενσωματωμένο Jetty web server, επιτρέποντας την τοπική διαχείριση αρχείων και την ταυτόχρονη πρόσβαση μέσω απομακρυσμένου web dashboard.
